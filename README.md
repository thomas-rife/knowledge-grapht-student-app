# Knowledge Grapht - Student

![Knowledge Grapht Logo](/assets/images/KGLogo.png)

**Knowledge Grapht** (formerly **Codelingo**) is a platform where teachers can create and send spaced repetition exercises to their students and track their individual progress for each topic using the titual knowledge graph feature.

It is being developed as a student project at Loyola Marymount University (LMU), California, as part of the Applied Cognitive Technologies (ACT) Lab, under the supervision of [Dr. Andrew Forney](https://forns.lmu.build/about.html).

## Features

Knowledge Grapht consists of two separate user interfaces: an administrative web application where teachers to create, manage, and review their classes; and a mobile application where students to complete quizzes and track their personal progress. *This repository only contains the mobile app that acts as the student interface.*
